### Editar Cliente

Para editar un cliente, accede a la sección de **Clientes** desde el menú principal. Busca al cliente que deseas editar y haz clic en **Editar Cliente**. Se abrirá un formulario donde podrás modificar la información de ese cliente.

#### Información Obligatoria:
- **Nombre**
- **Apellido**
- **Email**
- **Teléfono**

#### Información Adicional

Además, podrás editar o completar los siguientes campos opcionales, especialmente si eres administrador o superadministrador:

- **Empresa**
- **DNI/CUIL/CUIT/Pasaporte** (si aplica)
- **Dirección** (importante para colocaciones, envíos, etc.)
- **Valor de descuento**
- **Días de vencimiento de Cuenta Corriente** (si corresponde)
- **Tipo de usuario**, que puede ser uno de los siguientes:
  - Editor
  - Administrador
  - Galpón
  - Taller
  - Cliente
  - Cliente Cuenta Corriente
- **Incluir o no IVA**

Recuerda que solo los administradores o superadministradores tienen acceso para modificar estos campos adicionales.
