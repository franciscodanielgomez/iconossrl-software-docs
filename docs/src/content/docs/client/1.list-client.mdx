---
title: Gestión de Clientes/Usuario
description: Funcionalidades y uso del módulo de clientes
---

El módulo de clientes permite administrar toda la base de contactos comerciales del sistema. Desde aquí podrás:

- Registrar nuevos clientes.  
- Editar información existente.  

Todos los cambios se sincronizan en tiempo real con los demás módulos (Pedidos, Taller, etc.).  

### Cómo acceder  
1. Selecciona **Clientes** en el menú principal.  
2. Verás el listado completo.  
3. Usa el buscador superior para filtrar por **nombre, apellido, email o teléfono**.  

### Información en la tabla  
En la tabla de clientes verás las siguientes columnas:  

- **Nombre**  
- **Apellido**  
- **Empresa**  
- **Email / Teléfono**  
- **Rol** (Cliente, Editor, Administrador, etc.)  
- **Estado**  
- **Acciones**:  
  1. Crear presupuesto.  
  2. Ver historial de pedidos.  
  3. Editar cliente.  
  4. Crear pedido.  

En la parte superior de la tabla encontrarás un botón para **crear un nuevo cliente o usuario**.
