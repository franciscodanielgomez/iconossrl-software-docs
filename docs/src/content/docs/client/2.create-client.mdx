### Crear un Cliente

Para crear un cliente, debes acceder a la sección de **Clientes** desde el menú principal. Una vez allí, verás un botón que dice **Crear Cliente**. Al hacer clic en este botón, se abrirá un modal donde podrás ingresar los datos del usuario y/o cliente.

#### Datos Obligatorios:
- **Nombre**
- **Apellido**
- **Empresa** (opcional)
- **DNI/CUIL/CUIT/Pasaporte**
- **Email**
- **Teléfono**

### Crear un Nuevo Empleado

Si eres administrador y deseas agregar un nuevo empleado, primero deberás cargar al empleado como cliente utilizando los mismos pasos anteriores. Una vez creado como cliente, podrás editar su información en la sección **Editar Cliente** para convertirlo en un empleado.

Si deseas editar la información de un cliente después de haberlo creado, puedes buscarlo en la sección **Editar Cliente**.
