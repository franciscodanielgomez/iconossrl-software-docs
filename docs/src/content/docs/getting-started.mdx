[Iconos SRL](https://iconossrl.com.ar) es una solución integral para la gestión comercial y productiva, diseñada para optimizar procesos de ventas, producción y administración. Desarrollado por [Allianzy Inc](https://allianzy.com), líder en desarrollo de herramientas tecnológicas para negocios.

Esta guía proporciona instrucciones detalladas para utilizar todas las funcionalidades del sistema, incluyendo:
- Procesos paso a paso
- Mejores prácticas
- Soluciones a problemas comunes
- Configuraciones avanzadas

Para asistencia adicional, contacte a nuestro equipo de soporte técnico.

## Inicio Rápido

### Acceso al Sistema

Ingrese mediante los siguientes enlaces según su sucursal:

- **Sucursal Independencia:**  
  [https://iconosindependencia.flutterflow.app](https://iconosindependencia.flutterflow.app)
- **Sucursal Chacabuco:**  
  [https://iconoschacabuco.flutterflow.app](https://iconoschacabuco.flutterflow.app)
- **Ecommerce:**  
  [https://iconosecommerce.flutterflow.app](https://iconosecommerce.flutterflow.app)

### Cuenta de Administrador

Los propietarios y administradores recibirán credenciales con acceso completo al sistema, permitiendo:
- Gestión de usuarios
- Asignación de roles
- Configuración global

### Roles del Sistema

| Rol | Permisos |
|------|----------|
| **Superadministrador** | Control total del sistema |
| **Administrador** | Gestión completa de un local |
| **Editor** | Creación de pedidos, gestión de clientes y cajas |
| **Taller** | Gestión de órdenes de producción en taller |
| **Galpón** | Administración de trabajos en depósito |
| **Librería** | Acceso limitado a funciones comerciales |
