---
title: Crear cuenta
description: Guía para la creación y gestión de cuentas en el sistema.
---

Solo los usuarios con permisos de **Superadministrador, Administrador o Editor** pueden crear cuentas en la sección de **Clientes**.  

Para crear una cuenta a tu nombre, debes solicitar autorización al equipo encargado, quienes se ocuparán de gestionarla.  

## Información Requerida  

#### Para empleados  
Al crear una cuenta de empleado, se solicitarán los siguientes datos:  

- **Nombre**  
- **Apellido**  
- **Email**  
- **Teléfono**  
- **DNI/CUIL/CUIT/Pasaporte**  
- **Dirección**  

#### Para clientes  
Para registrar un cliente, es obligatorio proporcionar:  

- **Nombre**  
- **Apellido**  
- **Email**  
- **Teléfono**  
- **DNI/CUIL/CUIT/Pasaporte**  

### Información Adicional  

Además, podrás editar o completar los siguientes campos opcionales:  

- **Empresa**  
- **DNI/CUIL/CUIT/Pasaporte** (si aplica)  
- **Dirección** (importante para colocaciones, envíos, etc.)  
- **Valor de descuento**  
- **Días de vencimiento de Cuenta Corriente** (si corresponde)  
- **Tipo de usuario**, que puede ser:  
  - Editor  
  - Administrador  
  - Galpón  
  - Taller  
  - Cliente  
  - Cliente Cuenta Corriente  
- **Incluir o no IVA**
